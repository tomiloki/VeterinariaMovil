# Estructura del agente para MascotaFeliz

Esta carpeta define una estructura de trabajo para agentes de desarrollo enfocada en publicacion de portafolio, seguridad y mantenimiento continuo.

## Objetivo

Mantener una forma consistente de trabajar el proyecto, evitando fugas de secretos y asegurando calidad minima antes de cada push.

## Contenido

- `contexto-proyecto.md`: contexto tecnico y decisiones base del proyecto.
- `workflow.md`: flujo operativo por sesion.
- `checklists/publicacion-github.md`: checklist para dejar el repo listo para GitHub.

## Uso recomendado por sesion

1. Leer `contexto-proyecto.md`.
2. Seguir `workflow.md`.
3. Ejecutar checklist de publicacion antes de push.
