# AGENTS - MascotaFeliz

Este repositorio usa la carpeta `.agents/` como fuente de verdad para el trabajo asistido por agentes.

## Punto de entrada

1. Leer `.agents/contexto-proyecto.md`.
2. Seguir `.agents/workflow.md`.
3. Ejecutar `.agents/checklists/publicacion-github.md` antes de push.

## Principios

- No hardcodear secretos.
- Usar variables de entorno y `.env.example`.
- Mantener documentacion y setup actualizados.
